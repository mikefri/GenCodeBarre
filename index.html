❓ Guide d'utilisation GenCodeBarre v2.3

🛠️ Onglet Éditeur (Générateur)

  • Titre de page : Ajoute un titre qui ce positionnera en haut du code barre (ex: "Inventaire Magasin A").
  • Données (Code) : C'est ici que vous saisissez le texte ou le numéro qui sera encodé dans le code barre.
  • Type de Code : Choisissez le type de code (Code 128 pour la plupart des usages, EAN-13 pour les produits, QR Code pour les liens/infos).
  • Échelle du Code : Ajustez la taille physique du code-barres sur la page imprimée.
