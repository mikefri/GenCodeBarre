❓ Guide d'utilisation GenCodeBarre v2.4

📂 Onglet Planches

Cet onglet vous permet d'importer une liste de codes depuis un fichier Excel (.xlsx) afin de générer une ou plusieurs planche(s) complète(s) en une seule fois.

  • Source de Données : Assurez-vous que tous vos codes sont listés dans la colonne A du fichier Excel.
  • Type de Code : Choisissez le type de code (Code 128 pour la plupart des usages, EAN-13 pour les produits, QR Code pour les liens/infos).
  • Configuration Grille : Sélectionnez le type de planche que vous souhaitez.
  • Option Flèche latérale : Vous avez la possibilité d'ajouter une flèche d'orientation sur la partie gauche du code-barres.
  • Taille du texte : Ajustez indépendamment la taille du texte affiché sous le code-barres.
  • Texte personnalisé : Ajoutez un libellé fixe sur toutes les étiquettes (ex : nom de lot, référence).
  • Codes invalides : Les codes incompatibles avec le type choisi sont affichés en rouge avec un avertissement.
  • Copier un code : Cliquez sur une étiquette dans l'aperçu pour copier son code dans le presse-papier.
  • Réordonner : Glissez-déposez les lignes de saisie manuelle pour les réordonner.
  • PDF : Le bouton "Télécharger le PDF" génère un PDF multi-pages de toutes vos planches.
